## Simulated Enterprise Scenario

This project is based on a simulated enterprise environment representing a large financial institution, Cymbal Bank.

Cymbal Bank is a nationwide retail bank offering digital banking services built on a modern cloud-based infrastructure. As part of its digital transformation, the organization aims to strengthen its cloud security posture and ensure compliance with industry standards.

As a Cloud Security Engineer, the objective was to secure Cymbal Bank’s Google Cloud environment by leveraging Security Command Center (SCC) to:

- Identify security vulnerabilities and misconfigurations  
- Analyze high severity findings  
- Implement mitigation strategies  
- Improve visibility into security risks  

The environment included exposed services, disabled logging configurations, and potential application vulnerabilities that required investigation and remediation.

This scenario reflects real-world responsibilities of a security operations team working to detect and respond to threats in cloud environments.
